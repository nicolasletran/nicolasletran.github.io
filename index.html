Calories Intake Tracker

A Python command-line application for tracking daily meals, calories, and protein intake. Provides daily and weekly summaries and checks whether the user meets a personalized protein goal based on body weight.

Features:

  • Add meals: Food name, Calories, Protein (grams), Category (Breakfast, Lunch, Dinner, Snack)
  • View today’s summary and last 7 days summary
  • Edit or delete meals from any date
  • Personalized protein goal: 2 × body weight (kg) with visual feedback ✅ / ⚠️
  • Data stored locally using JSON

Built to demonstrate data handling, control flow, and clean program structure in Python.

CLI File Manager

A Command-Line Interface (CLI) File Manager implemented in C. Allows users to interact with the file system using commands similar to Unix shell.

Features:

  • List files and directories (ls, ls -l)
  • Create / Delete files (touch, rm)
  • Create / Delete directories (mkdir, rmdir)
  • Copy / Move files (cp, mv)
  • Recursive file search (find)
  • Navigation: cd and pwd
  • Color-coded directories and optional detailed listing

Built to demonstrate file system manipulation, recursion, and cross-platform C programming.
